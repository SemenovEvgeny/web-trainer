# Система управления тренировками

Веб-приложение для работы тренера с подопечными, включающее прием заданий и оценку качества решений.

## Возможности

### Для тренера:
- Просмотр списка подопечных с статистикой
- Создание заданий для подопечных
- Просмотр всех заданий с фильтрацией
- Проверка решений подопечных
- Оценка качества решений (Отлично, Хорошо, Удовлетворительно, Требует улучшения)
- Написание обратной связи

### Для подопечного:
- Просмотр списка своих заданий
- Просмотр деталей задания
- Отправка решений
- Просмотр оценок и обратной связи от тренера

## Технологии

- React 18
- TypeScript
- Vite
- Tailwind CSS
- React Router
- Lucide React (иконки)

## Установка и запуск

### Требования

Для работы приложения необходимо установить **Node.js** (версия 18 или выше).

#### Установка Node.js:

**macOS:**
- Скачайте установщик с официального сайта: https://nodejs.org/
- Или используйте Homebrew:
  ```bash
  brew install node
  ```

**Проверка установки:**
```bash
node --version
npm --version
```

### Шаги запуска:

1. **Установите зависимости:**
   ```bash
   npm install
   ```

2. **Запустите dev-сервер:**
   ```bash
   npm run dev
   ```

3. **Откройте браузер:**
   - Приложение будет доступно по адресу `http://localhost:5173`
   - Vite автоматически откроет браузер, или скопируйте URL из терминала

## Использование

1. На странице входа введите ваше имя и выберите роль (Тренер или Подопечный)
2. Войдите в систему

### Как тренер:
- На странице "Подопечные" вы увидите список всех подопечных с их статистикой
- На странице "Задания" можно просмотреть все задания и отфильтровать их
- На странице "Создать задание" создайте новое задание для подопечного
- Откройте задание, чтобы проверить решение и поставить оценку

### Как подопечный:
- На странице "Мои задания" просмотрите список ваших заданий
- Откройте задание, чтобы увидеть описание и отправить решение
- После проверки тренером вы увидите оценку и обратную связь

## Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут в папке `dist`.

### Быстрый старт с reg.ru:

**Вариант 1: Автоматический деплой (рекомендуется)**
```bash
# 1. Настройте конфигурацию
cp deploy-config.example .deploy-config
# Отредактируйте .deploy-config с вашими данными

# 2. Запустите деплой
npm run deploy:ftp
# или через SSH
npm run deploy:ssh
```

**Вариант 2: Ручной деплой**
```bash
# 1. Соберите проект
npm run deploy

# 2. Загрузите файлы из папки dist на сервер reg.ru через FTP/SSH
# 3. Загрузите .htaccess.example как .htaccess в корень public_html
```
